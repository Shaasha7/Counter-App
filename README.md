# Counter-App
🚀 Elite Counter Pro - A premium interactive counter dashboard built using HTML, CSS, and JavaScript with glassmorphism UI, animations, local storage, and smart DOM interactions.
# ⚡ Elite Counter Pro

![Elite Counter Pro Banner](https://img.shields.io/badge/Project-Elite%20Counter%20Pro-purple?style=for-the-badge)

## 🚀 Overview

**Elite Counter Pro** is a premium interactive counter dashboard built using **HTML, CSS, and JavaScript**.  

Instead of a traditional basic counter application, this project provides a modern dashboard experience with a luxury glassmorphism interface, dynamic animations, smart controls, and browser-based data persistence.

The project demonstrates the power of **JavaScript DOM manipulation** by converting a simple counter into a professional-level web application.

---

# ✨ Features

## 🎨 Premium User Interface
- Modern glassmorphism design
- Full-screen luxury dashboard layout
- Animated gradient background
- Smooth hover effects and transitions
- Responsive design

## ⚡ Counter Management
- Increment counter (+)
- Decrement counter (-)
- Reset counter
- Custom increment value selection

## 🧠 Smart Features
- 💾 Local Storage support
    - Saves counter value even after refreshing the page
- ⌨ Keyboard shortcuts
    - `+` → Increase counter
    - `-` → Decrease counter
    - `R` → Reset counter
- 🕒 Real-time date and time display
- 📊 Activity history tracking

## 📈 Dynamic Visualization
- Animated circular progress indicator
- Live counter updates
- Interactive dashboard components
